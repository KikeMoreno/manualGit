Instalación & Configuración de GitHub

INSTALACIÓN

Linux y Unix

En Debian/Ubuntu :
$ apt-get install git
En Fedora:
$ yum install git
En Gentoo:
$ emerge --ask --verbose dev-vcs/git
Arch Linux
$ pacman -S git
openSUSE
$ zypper install git
FreeBSD
$ cd/usr/ports/devel/git
$ make install
Solaris 11 Express
$ pkg install developer/versioning/git
OpenBSD
$ pkg_add git

Mac Os X

Mediante el enlace siguiente:
http://sourceforge.net/projects/git-osx-installer/
Mediante brew:
-Instalar Homebrew: http://brew.sh
-Comandos:
$ brew update
$ brew install git
Interfaz gráfica:
https://mac.github.com

Windows

Mediante uno de estos enlaces: http://git-scm.com/download/win
https://windows.github.com

CONFIGURACIÓN

Inmediatamente después de instalar Git, lo primero que deberías hacer es lanzar un par de comandos de configuración. Git trae una herramienta denominada git config que permite obtener y establecer variables de configuración.

Tu identidad

Para establecer tu nombre de usuario y dirección de correo electrónico.
$ git config --global user.name "name"
$ git config --global user.email "email@example.com"
Con la opción –global sólo necesitarás hacer esto una vez.

Comprobar la configuración

Para poder comprobar la configuración, puedes usar el comando git config --list, en el cual se listarán todas las propiedades que Git ha configurado:
git config --list
filter.lfs.clean=git lfs clean %f
filter.lfs.smudge=git lfs smudge %f
filter.lfs.required=true
user.name=name
user.email=email@example.com
