Instalación de GitHub

Linux y Unix

En Debian/Ubuntu :
$ apt-get install git
En Fedora:
$ yum install git
En Gentoo:
$ emerge --ask --verbose dev-vcs/git
Arch Linux
$ pacman -S git
openSUSE
$ zypper install git
FreeBSD
$ cd/usr/ports/devel/git
$ make install
Solaris 11 Express
$ pkg install developer/versioning/git
OpenBSD
$ pkg_add git

Mac Os X

Mediante el enlace siguiente:
http://sourceforge.net/projects/git-osx-installer/
Mediante brew:
-Instalar Homebrew: http://brew.sh
-Comandos:
$ brew update
$ brew install git
Interfaz gráfica:
https://mac.github.com

Windows

Mediante uno de estos enlaces: http://git-scm.com/download/win
https://windows.github.com
